
# 「它香 (PetScent)」网页生成超级指令 (Master Prompt)

## 1. 核心视觉 (Core Visuals)
- **风格**: 极简主义、人文关怀、实验室精密感、去人类中心化。
- **配色**: 燕麦白 (#FDFBF7) 背景，鼠尾草绿 (#829682) 装饰，森林绿 (#1B241B) 文字，精华金 (#D4B982) 强调。
- **特效**: 玻璃拟态 (Glassmorphism)，全站 15px 高斯模糊遮罩，容器圆角 3.5rem - 4rem。
- **交互**: 文字逐行淡入 (Fade-in-up)，图像采用“呼吸感”缓慢放缩，模拟空气流动。

## 2. 深度哲学：去人类中心化 (De-humanized Philosophy)
- **核心逻辑**: 网页需体现“生物本能优先”。不以人类的香气好恶为标准，而是以宠物的边缘系统安全为准则。
- **视觉映射**: 
  - 使用低饱和度、偏冷灰或燕麦色调的自然影像。
  - 大量留白 (Negative Space)，体现空气感与克制。
  - 按钮与装饰件采用极其细致的线条（0.5px - 1px）。
- **核心金句**: “真正的尊重，是当它们闭上眼，能嗅到家以外的整片旷野。”

## 3. 页面流 (User Flow)
1. **Hero**: 情感共鸣 -> 唤起对大自然的感知 -> 点击[获取建议]。
2. **Philosophy**: 建立信任 -> 重点展示“去人类中心化”的四个准则与科学背书卡片。
3. **Products**: 解决问题 -> 展示四大气味方案：雷雨安（应激）、暂别安（焦虑）、出行安（调节）、窝窝安（深睡）。
4. **AI Labs**: 记忆实验室 -> 上传照片，应用预设滤镜（高级感胶片、温暖夕阳等）。
5. **AI Consultant**: 智能诊断 -> 基于行为描述提供分子干预方案。
6. **Footer**: 品牌落地 -> Slogan: "因为爱，所以“它香”。LOVE, SCENTED NATURALLY."

## 4. 产品系列细节
- **雷雨安 (Storm)**: 抗应激。针对巨响。成分：高地薰衣草、罗马洋甘菊、岩兰草。
- **暂别安 (Absence)**: 长效舒缓。针对分离焦虑。成分：苦橙叶、马郁兰、喜马拉雅雪松。
- **出行安 (Road)**: 即时调节。针对环境变换。成分：生姜、甜罗勒、乳香。
- **窝窝安 (Sleep)**: 深睡修复。针对老年宠物。成分：橙花、缬草根、檀香。

## 5. Logo 逻辑
- 顶部单片叶片（大自然）+ 水滴框（萃取）+ 人/犬/猫嵌套（跨物种和谐）。线条宽度一致，体现平衡感。
