
# 它香 (PetScent) - 跨物种友好芳疗品牌官网构建指令

## 1. 品牌愿景与核心理念
- **品牌名称**: 它香 (PetScent / PetAroma)
- **品牌定位**: 中国首个以宠物嗅觉安全为核心的科学芳疗品牌。
- **核心关键词**: 去人类中心化、生物本能、医疗级安全、情绪愈合、极简美学。
- **视觉风格**: 像“燕麦奶”一样温润，像“森林”一样深邃，充满高级感与宁静感。

## 2. 视觉规范 (Design System)
### 2.1 色彩方案 (Palette)
- **底色 (Oatmeal)**: `#F9F7F2` - 温暖的自然底色。
- **主色 (Sage)**: `#B4C3B4` - 柔和的鼠尾草绿。
- **点缀色 (Forest)**: `#2D3A2D` - 沉稳的森林深绿（用于文字和强调）。
- **辅助色 (Essence)**: `#E6D5B8` - 植物萃取的质感金。
- **氛围色 (Petal)**: `#F2E3E3` - 极浅的瓣粉。

### 2.2 字体规范
- **中文**: Noto Serif SC (思源宋体) - 呈现品牌的人文厚度与科学严谨。
- **英文**: Montserrat - 呈现现代感。
- **排版要求**: 极大的字间距 (Tracking)，极致的留白。

### 2.3 动态效果
- **呼吸感 (Breathe)**: 所有的背景光晕和重要组件需有极慢的透明度/缩放呼吸动画。
- **优雅入场**: 页面滚动时，文字和卡片需以 `cubic-bezier` 曲线向上滑入。

## 3. 页面结构与组件逻辑
### 3.1 导航栏 (Header)
- 玻璃悬浮质感 (`backdrop-filter: blur`)。
- 包含“芳疗哲学”、“气味方案”、“视觉实验室”、“智能诊断”。
- 隐藏彩蛋：控制台终端入口。

### 3.2 英雄区 (Hero Section)
- **意境图**: 呈现宠物在自然光影下的安宁瞬间。
- **文案**: “每一缕清香，都是大自然对它的轻声细语。”
- **交互**: 漂浮的植物分子光晕。

### 3.3 品牌哲学 (Philosophy)
- 强调 0% 酚、0% 醇。
- 对比表：展示普通香氛与“它香”科学准则的差异。

### 3.4 产品矩阵 (Product Grid)
- **四款核心产品**: 雷雨安 (Storm)、离别安 (Alone)、出行安 (Road)、窝窝安 (Sleep)。
- **交互**: 鼠标悬停时，展示“前/中/后”三段式气味金字塔。

### 3.5 AI 智能诊断 (AI Consultant)
- **功能**: 用户输入宠物类型和焦虑行为。
- **逻辑**: 调用 Gemini API 生成专业的行为诱因拆解、气味干预建议及环境优化方案。

### 3.6 视觉记忆实验室 (Image Lab)
- **功能**: 用户上传爱宠照片，输入修改意向（如“电影质感”）。
- **逻辑**: 调用 Gemini 2.5 Flash Image 重新生成具备高艺术感的图像。

### 3.7 实验室控制台 (Terminal)
- **功能**: 全代码驱动的交互，输入 `analyze` 查看分子结构。

## 4. 技术栈要求
- **框架**: React 19 + TypeScript。
- **样式**: Tailwind CSS。
- **API**: 使用 `@google/genai` (Gemini-3-flash 用于咨询，Gemini-2.5-flash-image 用于图片编辑)。
- **环境**: 需要配置 `process.env.API_KEY`。

## 5. 交互细节建议
- 所有按钮在悬停时需有轻微的位移和阴影增强。
- 页面滚动条需定制为极细的鼠尾草绿线条。
- 加载状态（Loading）需描述为“正在萃取影像精华”或“正在调配分子方案”。
