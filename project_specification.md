
# 「它香 · PetScent」品牌数字化平台需求规格说明书

## 1. 品牌愿景与核心哲学
- **定位**: 中国首个去人类中心化宠物芳疗品牌。
- **哲学**: 尊重本能（Instinct First）、临床背书（Clinically Backed）、跨物种共情（Interspecies Empathy）。

## 2. 视觉规范 (Visual Identity)
### 2.1 配色方案 (Color System)
- `Oatmeal (#FDFBF7)`: 主背景，象征纯净与安全。
- `Sage (#829682)`: 品牌色，传递自然疗愈感。
- `Forest (#1B241B)`: 核心文字，代表权威与沉稳。
- `Essence (#D4B982)`: 点缀色，代表萃取精华。

### 2.2 视觉元素
- **超大圆角**: `3.5rem - 4rem`，弱化工业感，增强亲和力。
- **动态呼吸感**: 核心背景图采用缓慢缩放动画，模拟生命节律。
- **极细线条**: `0.5px` 分隔线，体现精密芳疗逻辑。

## 3. 功能特性实现
### 3.1 AI 视觉实验室 (Image Alchemy)
- **输入**: 宠物照片 + 意境关键词。
- **逻辑**: 通过 Gemini 2.5 Flash Image 引擎，强制全彩输出，强化暖调与电影质感。
- **输出**: 高清炼金处理后的宠物记忆影像。

### 3.2 AI 深度咨询师 (Reasoning Advisor)
- **底层逻辑**: 引入思维链 (Chain of Thought) 推理。
- **诊断路径**: 物种特性 -> 应激诱因 -> 神经干预 -> 分子处方。
- **用户体验**: 结构化的临床报告展示，提供具体的环境补偿建议。

### 3.3 四季气味图谱 (Product Matrix)
- **产品分类**: 雷雨安、暂别安、出行安、窝窝安。
- **交互详情**: 交互式展示前/中/后调成分，实现成分透明化。

## 4. 技术栈
- **框架**: React 19 + TypeScript。
- **AI**: Gemini API (Flash/Pro 混合调用)。
- **样式**: Tailwind CSS (移动端适配优先)。
- **交互**: Framer Motion 风格的 CSS 动画。

## 5. 市场适配 (China Market Optimization)
- **交互语境**: 符合中文语境的“专业、尊重、温和”风格。
- **物种支持**: 涵盖猫、狗、兔、仓鼠、鸟类、爬宠。
- **分享友好**: 针对移动端浏览器与社交应用内置浏览器的布局优化。
